Key Interactions

  • Drag & reorder: Operations can be reordered within each setup via drag handles (⠿)
  • Inline edit: Click ✏️ to expand operation details (speeds, feeds, depth of cut, etc.)
  • Version control: Each Save or Copilot change creates a new version; switch via dropdown
  • Dirty state: Warning shown in chat if user has unsaved manual edits before Copilot makes changes
  • PDF export: Generates a formatted process sheet with all operations, tools, and parameters
